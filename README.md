Web Turistička Agencija
Web aplikacija turističke agencije razvijena u ASP.NET MVC tehnologiji, namenjena upravljanju turističkim aranžmanima, rezervacijama i korisnicima, sa jasno definisanim korisničkim ulogama.

Tehnologije

Backend: ASP.NET MVC, C#
Frontend: HTML, CSS, JavaScript
Arhitektura: MVC (Model-View-Controller)
Rad sa podacima: Čitanje i upis podataka iz tekstualnih fajlova
Autentifikacija i autorizacija: Uloge i kontrola pristupa
Kontrola verzija: Git

Arhitektura sistema

Aplikacija je realizovana po MVC arhitektonskom obrascu:

Model – poslovna logika i rad sa podacima
View – korisnički interfejs
Controller – obrada zahteva i koordinacija između modela i pogleda
Poseban akcenat stavljen je na razdvajanje odgovornosti i čitljivost koda.
